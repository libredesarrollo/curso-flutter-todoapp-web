# Todo App - Primeros Pasos con Flutter

![Flutter](https://img.shields.io/badge/Flutter-02569B?style=for-the-badge&logo=flutter&logoColor=white)
![Dart](https://img.shields.io/badge/Dart-0175C2?style=for-the-badge&logo=dart&logoColor=white)

Esta es una aplicación de gestión de tareas (Todo App) desarrollada como parte del aprendizaje práctico del ecosistema Flutter. El proyecto implementa funcionalidades fundamentales para cualquier aplicación móvil moderna, incluyendo manejo de estados, persistencia básica y componentes de interfaz de usuario avanzados.

## 📚 Libro de Referencia

Este proyecto sigue las lecciones y mejores prácticas detalladas en el libro:

**[Primeros pasos con Flutter](https://www.desarrollolibre.net/libros/primeros-pasos-flutter)**

Un recurso completo para dominar el desarrollo de aplicaciones móviles multiplataforma desde cero.

## 🚀 Características

- **Gestión de Tareas:** Creación, visualización y organización de actividades diarias.
- **Selección de Fechas:** Integración con `syncfusion_flutter_datepicker` para una experiencia de usuario fluida al programar tareas.
- **Animaciones Modernas:** Uso de `delayed_display` para transiciones suaves en la interfaz.
- **Manejo de Imágenes:** Integración con `image_picker` para adjuntar contenido visual a las tareas.
- **Diseño Responsive:** Interfaz adaptativa construida con los widgets de Material Design.

## 🛠️ Tecnologías Utilizadas

- **Flutter SDK:** ^3.12.0
- **Paquetes Principales:**
  - `syncfusion_flutter_datepicker`
  - `delayed_display`
  - `image_picker`
  - `path_provider`

## 📦 Instalación y Configuración

Para ejecutar este proyecto localmente, asegúrate de tener configurado el entorno de desarrollo de Flutter.

1.  **Clonar el repositorio:**
    ```bash
    git clone [url-del-repositorio]
    ```
2.  **Instalar dependencias:**
    ```bash
    flutter pub get
    ```
3.  **Ejecutar la aplicación:**
    ```bash
    flutter run
    ```
